Project name: Paint By Numbers
Project description: The program will generate the Paint by Numbers version of the input image and allow the user to download the generated image. Also, it features a graphical interface where user can paint the generated output and other coloring images digitally and save that digital painting.
How to RUN:
Note: PIL and tkinter modules must be installed
Download the folder PBN, open main.py, run. Insert your name in the entry box and press PBN generator if you want to generate the PBN image or press Start Coloring if you want to paint on the image. Most of the buttons and widgets on screen are accompanied by tooltips.
